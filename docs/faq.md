# FAQ

## Do I need the MCP server to use the skills?

No. The repo can be used as a skill marketplace and exporter without running the MCP server.

## Why use GitHub Pages instead of the wiki?

GitHub Pages gives a better landing experience, versioned docs, stronger SEO, and a cleaner public surface than a separate wiki.

## Why both `skill.yaml` and `SKILL.md`?

`skill.yaml` fits orchestrator-mcp and structured registries. `SKILL.md` is more portable across agent ecosystems that use markdown-based skill packs.

## Why recommend Ollama as the cheapest default?

Because local Ollama or Ollama Pro usually gives the strongest cost-to-capability ratio for repeated coding work, while cloud providers remain available as specialist fallback.
