# Providers

## Supported providers

- Ollama
- Gemini
- NVIDIA-hosted OpenAI-compatible models

## Recommended setup

- Start with Ollama or Ollama Pro for cheap default capacity
- Add Gemini for multimodal review
- Add NVIDIA only when you want extra specialist breadth

## Environment variables

- `OLLAMA_API_KEY`
- `OLLAMA_BASE_URL`
- `GEMINI_API_KEY`
- `NVIDIA_API_KEY`
